Порядок действий:

1. Открываем скрипт script1 (создание инфоблоков).php
Копируем оттуда код без тегов php <? и ?>

2. На своём проекте Битрикс в админке открываем Инструмент Командная PHP-строка (/bitrix/admin/php_command_line.php?lang=ru#authorize)

3. Вставляем скопированный код и нажимаем "Выполнить"

4. После выполнения скрипта добавляется тип инфоблока Офисы (/bitrix/admin/iblock_type_admin.php?lang=ru)

5. В настройках созданного инфоблока смотрим ID свойств:
- ID самого инфоблока 
- ID поля Название офиса
- ID поля Телефон
- ID поля Email
- ID поля Координаты
- ID поля Город

6. В настройках созданного инфоблока переходим во вкладку Доступ и ставим доступ по умолчанию для всех "Чтение"

7. В папке moduleGpnOffices правим файл scriptAddOffices.php, добавляя свои ID из пятого пункта

8. Копируем папку moduleGpnOffices в корневой раздел своего проекта

9. Открываем браузер и переходим по ссылке ВАШ_ПРОЕКТ/moduleGpnOffices/scriptAddOffices.php 

10. После отрабатывания скрипта создаются элементы инфоблока Офисы:
ГПН-Цифровые Решения, Санкт-Петербург, Киевская 5,
ГПН, Санкт-Петербург, Почтамтская, 3-5
ГПН-Цифровые Решения, Москва, Научный 17
Газпром нефть шельф, Платформа Приразломная

11. Проверяем свойства элементов и наличие детальной картинки (вкладка Подробно)

12. На вашем проекте в папку /local/components/ копируем папку gpn

13. Правим файл offices.php, добавляя свой ID инфоблока

14. На ваш проект в корневой раздел копируем файл offices.php

15. Открываем браузер и переходим по ссылке ВАШ_ПРОЕКТ/offices.php

16. Не забудьте удалить с вашего проекта папку moduleGpnOffices
